Syntax Hightlight
=================

1. set syntax language for current file

  Show syntax/filetype used for current file:

  ```shell
  :set syntax
  :set filetype
  ```

  Set syntax/filetype for current file:

  ```shell
  # set one of the following parameter to highlight current file
  :set syntax=erlang
  :set filetype=erlang
  ```
