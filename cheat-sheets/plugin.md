some plugin commands
==========================

## Nerdtree

| Command | Function |
| :------ | :-------- |
| m  |  open the menu |
| s  | open a file in split format |
| t  | open a file in new tab |
| gt | jump between tabs |
| R  | refresh |
| CD | come back the first vim path |
| cd | change to new path |
